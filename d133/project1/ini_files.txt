0000.ini - образец файла для новых приборов.
Стремимся избегать прошивать лишние параметры - параметры соответсвующие
дефолтным в dda_config.h. Лишние параметры - лишнее время доступа.
Всего прибор поддерживает таблицу параметров размером 32 
(см. dda_settings.c - MAX_SETTING_COUNT). 


==== Формат ini файлов, используемых для записи параметров в прибор.

1. Коментарием является символ # - по этому символу обрезается строка
2. Строка настроек считается валидной, если соответствует шаблону
<name> = <value>
Пробелы игнорируются. Если нужно дать строку с пробелами - использовать кавычки
<name> = "value 2"

==== Параметры, используемые прибором
serial = "2201"
Серийный номер - отображается на стартовой заставка

lang = 0
Язык индикации:0 -English 1-russian 

model = "DDA-133"
Модель - отображается на стартовой заставка

sensRatio =  0.245 
Соотношение усилия [Н] к дискретам АЦП

stepRatio = "0.0714286"
Микроны к шагам плунжера

cellStep = 2 
Шаг кассеты

cTimeout = 10000
Таймаут кассеты в мс

dTouch = 20 
Дискрет АЦП, которые считаются признаком касания

maxForce = 200
Максимальное усилие для датчика

slowRate = 128 
Скорость измерения и подхода к нижнему датчику 1-мин, 255 -макс
Это конечная точка в таблице разгона. 128 - примерно половинная скорость 

umSlow = 1500
Расстояние в мкм от нижнего датчика, которое плунжер проходит на максимальной скорости (при калибровке) 

pTimeout = 40000 
Таймаут плунжера в мс

slowDown = 2000 
Количество шагов перед подходом к нижнему датчику, после которого понижается скорость до slowRate 

Настройки шаговых двигателей:

minPeriod1 = 500 
Мин. период двигателя плунжера в мкс. 

minPeriod2 = 300
Мин. период двигателя кассеты в мкс. 

stopCount = 4 
Количество периодов по 30мс удержания. Длительност удержания = stoppageCount * 30 ms

accStep1 = 1 
Шаг в таблице разгона при ускорении двиг. плунжера
Acceleration step count = 256/accStep

decStep1 = 16 
Шаг в таблице разгона при замедлении двиг. плунжера
Deceleration step count = 256/decStep

chStep1 = 1 
Шаг в таблице разгона при изменении скорости двиг. плунжера
Step count = 256/chStep

Тоже самое, но для двигателя кассеты
accStep2 = 1
decStep2 = 4
chStep2 = 1
